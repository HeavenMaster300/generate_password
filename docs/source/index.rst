==================
Обзор проекта
==================

CLI-утилита для генерации безопасных паролей с гибкой настройкой символов и сохранением хэшей в JSON.

Функции:
- Генерация паролей (8–128 символов)
- Включение/исключение типов символов
- Хэширование (SHA-256)
- Сохранение по метке
- Поиск по метке

.. note::
   Пароли **не сохраняются в открытом виде** — только хэши!
